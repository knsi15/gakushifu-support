# がくカレ サポートページ

「がくカレ」は、大学の授業時間割とアルバイトのシフトをひとつのカレンダーで管理できるモバイルアプリです。

## アプリについて

| 項目 | 内容 |
|------|------|
| アプリ名 | がくカレ |
| 対応OS | iOS |
| 価格 | 無料 |
| 言語 | 日本語 |

## 主な機能

- 週間カレンダーで授業とシフトを一覧表示
- 曜日×時限の時間割グリッド
- シフトの登録・複製・まとめて登録
- バイト先ごとの色分け管理
- 学期ごとの時間割切り替え
- データのバックアップ・復元
- 完全オフライン動作（アカウント登録不要）

## よくある質問

### データはどこに保存されますか？
すべてのデータはお使いの端末内にのみ保存されます。外部のサーバーには一切送信されません。

### 機種変更時にデータを移行できますか？
「設定」→「バックアップ・復元」からデータをJSONファイルとして書き出し、新しい端末で読み込むことができます。

### アプリを削除するとデータはどうなりますか？
アプリをアンインストールすると、すべてのデータが端末から完全に削除されます。必要に応じて事前にバックアップを取ってください。

### アカウント登録は必要ですか？
不要です。インストール後すぐにすべての機能をご利用いただけます。
